현재 디자인된 홈페이지의 히어로 섹션을 업그레이드 해줘.

중요한 조건이 있다.

1. 기존 페이지의 전체 구조와 디자인 톤은 유지해야 한다.
2. 아래에 있는 카테고리 이미지 캐러셀, 뉴스, 파트너 영역과 자연스럽게 연결되어야 한다.
3. 디자인을 과하게 화려하게 만들지 말고, 지금의 미니멀하고 따뜻한 스타일을 유지해야 한다.
4. 영상 썸네일이나 복잡한 카드 UI는 추가하지 않는다.
5. 관광 포털이나 여행사 랜딩페이지처럼 보이면 안 된다.
6. 전체 인상은 "고급 로컬 플랫폼"이어야 한다.

현재 히어로 구조는 중앙 정렬 구조이다.

현재 구성

- 메인 헤드라인
- 설명
- 검색창
- 태그
- 통계

문제점

- 히어로 공간이 넓지만 정보 레이어가 부족해 비어 보인다.
- 플랫폼 서비스 특성상 탐색 UX가 부족하다.

그래서 히어로를 완전히 바꾸는 것이 아니라

**중앙형 hero → 좌우 split hero 구조로 업그레이드** 한다.
